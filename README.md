#### Running the app

**Describe what npm scripts do**
test
    runs jest tests
lint
    runs linter
test-watch
    runs jest tests that updates upon changing code
test-verbose
    runs jest tests with verbose
jsdoc
    creates docs using jsdoc
start
    runs the server
start:watch
    runs the server and restarts server upon changing code
